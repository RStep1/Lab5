help: выводит справку по доступным командам
info: выводит информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
show: выводит в стандартный поток вывода все элементы коллекции в строковом представлении
insert: добавляет новый элемент с заданным ключом
update: обновляет значение элемента коллекции, id которого равен заданному
remove_key: удаляет элемент из коллекции по его ключу
clear: очищает коллекцию
save: сохраняет коллекцию в файл
execute_script: считывает и исполняет скрипт из указанного файла. В скрипте содержатся команды в таком же виде, в котором их вводит пользователь в интерактивном режиме.
exit: завершает программу без сохранения в файл
remove_greater: удаляет из коллекции все элементы, превышающие заданный
remove_lower: удаляет из коллекции все элементы, меньшие, чем заданный
remove_greater_key: удаляет из коллекции все элементы, ключ которых больше или равен заданному
remove_all_by_engine_power:  удаляет из коллекции все элементы, значение поля enginePower которого эквивалентно заданному
count_by_fuel_type: выводит количество элементов, значение поля fuelType которых равно заданному
filter_less_than_fuel_type: выводит элементы, значение поля fuelType которых меньше заданного
